# Getting Started
---
## Clone Repository

```bash
git clone https://github.com/username/mindforge-template.git
````

Masuk ke folder project:

```bash
cd mindforge-template
```

---

## Install Dependencies

Karena Tailwind sudah dikonfigurasi, cukup jalankan:

```bash
npm install
```

---

## Run Tailwind Development Mode
```bash
npm run dev
```

Jika berhasil, akan muncul:

```
Rebuilding...
Done in XXXms
```

Biarkan terminal tetap berjalan selama development.

---

## Open Project

Buka file:

```
index.html
```
